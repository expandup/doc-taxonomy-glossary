Installation by WordPress

After downloading your package, please refrain from loading the entire package before extracting the file. The package is in ZIP format, and you can use an application or program for unpacking it (such as 7zip, WinRAR, WinZip, or others).

Once you've extracted the file, navigate to the resulting folder and look for a subfolder named 'plugin.' Inside this folder, you will find a file with the same name as the plugin you intend to install. This file is also in ZIP format and needs to be installed on your WordPress site.

To complete the installation, follow these steps:

  1. Log in to your WordPress website.
  2. Go to the 'Plugins' page in the WordPress Admin Panel.
  3. Select 'Add New Plugin.'
  4. On the plugin upload page, click on the 'Upload Plugin' button (or a similar option depending on your WordPress version).
  5. Browse for and select the ZIP file of the plugin you previously extracted.
  6. Click 'Install Now' and follow the instructions to activate the plugin on your WordPress site.

Please remember that plugin installation is a crucial step in managing your website, so follow these instructions carefully to ensure a successful integration.

Installation by FTP

Step 1: Preparation

Before you begin, make sure you have the ZIP file of the plugin you want to install on hand and unzipped. You will also need an FTP client such as FileZilla.

Step 2: Connect to Your Server via FTP

Start the FTP client (for example, FileZilla). Connect to your WordPress site's server using your FTP credentials (you'll usually find this information in your hosting provider's control panel).

Step 3: Navigate to the Plugins Folder

After connecting successfully, you will see a folder structure of your website on the right side of the FTP client. Navigate to the folder where WordPress is installed. This is usually done by going to the "public_html" folder or your website's root directory. Inside that folder, find the subfolder called "wp-content" and click to open it.

Step 4: Find the Plugins Folder

Inside the "wp-content" folder, you'll find a folder called "plugins." Click to access it.

Step 5: Upload the Plugin via FTP

On the left side of the FTP client, navigate to the folder where you unzipped the plugin file on your computer. Select all files and folders from the unzipped plugin. Drag these files and folders to the "plugins" folder in the FTP client (right side).

Step 6: Activate the Plugin in WordPress

After the files have been successfully transferred, access your WordPress site's admin panel. Go to the "Plugins" section in the menu. You should see the plugin you just downloaded listed among the available plugins. Click "Activate" under the plugin name.

Step 7: Configure the Plugin

License activation

If you are having trouble accessing the configuration options of your plugin or if it has stopped working, displaying messages like "Please activate the plugin license to access all features" in the administrative area of your WordPress, it means that you need to activate the plugin license.

Attention: The following procedure applies to all our plugins. To activate the plugin license, follow the steps below:

  1. Access your account page on the ExpandUP website and go to the "Application Password" tab. Create a password and store it securely, as it will be displayed only once. If you lose it, you will need to generate a new password for future activations.

  2. On the same account page, go to the "Library" tab and click on "Manage Domain" next to the purchased plugin. Now enter the desired domain to activate the plugin. After pasting the URL of your domain, click on "Generate License" to generate your license. Copy the provided code.

  3. On the site where the plugin is installed, access the plugin activation page. Enter your username registered on the ExpandUP website, the Application Password, and the generated License. Click "Save" and then "Activate Now." If all the information is correct, your plugin will be successfully activated.

FAQ

Do I need to generate an application password for each license activation?

No, you only need to generate a new application password if you lose the one generated previously.

Can I use the same application password on multiple sites?

Yes, you can use the same application password to activate any Expand UP product on any website you own; however, each license must be exclusive to each website and product.

Will my products that were activated with the previous password stop working when I generate a new application password?

No, once activated, the products remain active indefinitely.

Can I share my application password?

This is not recommended. Only share it with your team members or those responsible for maintaining your site. Mass disclosure of your application password is considered a serious violation, and your account may be suspended or canceled as a result.

Can I change or remove my application password?

Yes, you can do this at any time on your account page on the Expand UP website.

Can I activate Expand UP products in my testing environment?

Yes, simply follow the same step-by-step process to activate the product on your main domain. Once activated on your main domain, it becomes available for activation in testing and development environments as well. Use the same credentials to activate the product in your testing environments.
